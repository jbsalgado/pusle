# Walkthrough: Busca Inteligente e Scanner no Orçamento

Modernizamos a experiência de busca no módulo de Orçamentos para agilizar o atendimento.

## 🔍 Busca de Produtos
- **Inteligência:** Agora você pode buscar por qualquer parte do nome do produto, código de referência ou código de barras.
- **Acentuação:** Não se preocupe com acentos. "Café" e "Cafe" retornarão os mesmos resultados.
- **Múltiplos Termos:** Busque por "CIMENTO CP2" para encontrar exatamente o que deseja.

## 📷 Scanner via Webcam
- Clique no ícone de **Código de Barras** à direita da barra de pesquisa.
- Aponte o código para a câmera.
- O sistema lerá o código e filtrará o produto instantaneamente na grade.

## ⌨️ Leitores Físicos (USB/Bluetooth)
- O sistema agora escuta "bips" de scanners externos.
- Basta escanear o produto; o sistema preencherá a busca e filtrará o item por você.
